== README

Steps to run the app

1. Install Rails 4.2.4 with ruby version 2.2.0 preferred.

2. <tt> bundle exec rake db:create </tt>

3. <tt> bundle exec rake db:migrate </tt>

4. The postman collection link can be found at https://www.getpostman.com/collections/32917ce2f4038bce9bba

Question No. 2 is in Session_controller.rb
Question No. 4 is in beer_type_controller.rb
